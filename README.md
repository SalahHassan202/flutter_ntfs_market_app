# Artify â€“ NFT Marketplace UI

A clean and modern **NFT marketplace UI** built using **Flutter**, inspired by the Artify community Figma design.

This project is created for **learning and practice purposes**, with a strong focus on UI structure, layout, and reusable widgets.

---

## Screens

The application includes four main screens:

- Welcome / Onboarding  
- Home (NFT listings and collections)  
- Collection / Profile  
- NFT Details  

All screens are implemented using **pure Flutter UI**, without any business logic.

---

## Features

- Clean and modern interface  
- Pixel-perfect layout based on Figma  
- Responsive UI for different screen sizes  
- Smooth screen navigation  
- Organized widget structure  
- Local image assets  

---

## Tech Stack

- Flutter

- Material Design  
- Google Fonts  
- Local assets

## Getting Started

To run the project locally:

```bash
git clone https://github.com/SalahHassan202/flutter_ntfs_market_app.git
cd artify-nft-ui
flutter pub get
flutter run
````

---

## Project Scope

* UI only
* No backend
* No APIs
* No authentication
* No blockchain logic

This project is intended for learning, practice, and portfolio use.
